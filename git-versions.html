Gerenciando versões com GIT

Um projeto é criado no seu computador e salvo. Para iniciar o versionamento de código usando o GIT no seu computador dê um "Git Bash Here" na pasta do projeto e insira os comandos a seguir no terminal: 


git init

git add.     //prepara os arquivos na área stage - aqui ele irá selecionar todos//

git commit -m "Mensagem Explicativa da Versão"

git branch -M main     //para garantir que será salvo como Main - e não master//

git remote add origin git@github.com:seuusuario/seurepositorio.git     //linka o projeto local com o repositório GitHub

git push -u origin main     //envia o projeto para o GitHub

Quando preparamos o projeto para enviar, git add., podemos conferir tudo que está preparado para envio com o comando git status. Sempre que precisar checar o status do projeto com GIT, use esse comando.
